💳 Helcim Payment Integration
This POS integrates with Helcim for tap-to-pay and card payments.

No monthly fee · Interchange+ pricing (~1.5–2.5% per transaction) · Hardware $99–$329

Sign up at helcim.com, order a terminal, and paste your credentials below.

🔐 Security: Create a scoped API token in Helcim → API Access → New Token with only Payment: Write and Refund: Write permissions. Never use your full account token here.
⚠️ CORS Proxy: Browsers may block direct calls to api.helcim.com from a local file. If payments fail with a network/CORS error, deploy a simple proxy (free Cloudflare Worker or Vercel function) that forwards requests to https://api.helcim.com and enter its URL above. Leave blank to try direct connection first.
🖨️ Receipt Printer & Cash Drawer
Enter your receipt printer's IP address to automatically kick the cash drawer on cash sales.

• Star TSP143III LAN — check printer IP by holding Feed button on power-up (prints config ticket)
• Epson TM-T20III / TM-T88VI — same, or check router's connected devices list

The printer and iPad must be on the same Wi-Fi network.
📨 SMS & Email Receipts
Connect Twilio for SMS receipts and Resend for email receipts. Resend is free for up to 3,000 emails/month permanently.

• Twilio: twilio.com → Account SID + Auth Token + a purchased phone number
• Resend: resend.com → API Keys → Create API Key → Full Access

⚠️ These API calls also require a CORS proxy (same as Helcim above) or a small backend server.
💾 Backup & Restore
Export all your data (menu, employees, transactions, tabs) to a single JSON file. Restore from that file on any device to migrate or recover data.
🔄 Multi-Device Sync (Firebase)
Connect Firebase to sync tabs, transactions, employees and menu across all your iPads and devices in real time — for free.

How to set up (5 minutes):
1. Go to console.firebase.google.com and create a free project
2. Click Build → Realtime Database → Create Database (choose "Start in test mode")
3. Go to Project Settings → Your Apps → Add App → Web
4. Copy the firebaseConfig object and paste it below
5. Hit Save & Connect
